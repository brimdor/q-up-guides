Item Types

The 200+ items are divided into types that interact with set bonuses:

Warbanner
Battle-oriented. Counts toward "Same Type" sets.
Emoji / Quip
Expression types. Count toward "Typical" sets.
Emote
Duel type. Counts toward PVP set.
Weapon
Offense slot. Modifies QMult, trigger chance, passives.
Armor
Defense slot. Economic or defensive skills.
Trinket
Accessory slot. Often highest-scaling for Medic builds.

Rarity Tiers

Common
Base items. Low shop cost. Trash Collection set (+50%). Easy to bulk-buy.
Uncommon
Slightly stronger stats. Garage Sale set (+90%). Good mid-tier recycler fodder.
Rare
Strong passives. Estate Sale set (+150%). Gold-giving items cluster here.
Impossible
Highest rarity. Anomaly set (+300%). Extremely hard to collect.

Item Branches (for Timeline Saturated set)

  • Mech — Technology/cybernetic themed
  • Magic — Arcane/spiritual themed
  • Material — Physical/crafted resource themed
💡 Shards
Shards are inventory items separate from equipped gear. They don't take equipment slots but provide passive modifiers. Managing shard capacity is a key part of progression.

Best Practices

  1. Equip early, recycle late — Use items while they give QMult, then recycle when you outscale them.
  2. Sell doubles — Common duplicates are best recycled, not equipped.
  3. Save Impossible/Rare — Don't recycle high-rarity items unless making a full Anomaly set.
  4. Branch stack — Medic benefits strongly from Magic branch passives.
  5. Check Target Set — Before recycling, check which set is highlighted. Recycle around it.
