## Contributing

Want to add a guide?

1. Create an HTML file: `{slug}.html`
2. Add an entry to `guides.json`
3. Push to `main` branch — GitHub Pages auto-deploys within ~3 minutes
4. Verify on the live site

All guides use dark-mode CSS matching the in-game aesthetic.

## CI

Pull Requests touching `guides.json` or `*.html` automatically validate that every `guides.json` entry has a matching HTML file.
