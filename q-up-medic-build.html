Infinity QMult Medic — "Ride of the Valkyrie"
Achieves ∞ QMult every flip. Win or lose — doesn't matter.
✅ Why This Build Works
Medic's skill tree contains QMult-scaling nodes activating independently of wins/losses. By triggering 100% of Medic nodes, all QMult passives fire simultaneously, producing effectively infinite score growth.

Core Concept

Traditional builds rely on winning. The Infinity Medic build bypasses that. Score growth is decoupled from the 50/50 outcome and instead driven by QMult passives per flip.

Build Requirements

Hero: Medic
Requires unlock. Medic is a dedicated QMult archetype.
Full Skill Tree
All Medic nodes purchased. Takes many runs.
Optimal Placement
Nodes must be arranged to trigger geometrically. Adjacent QMult nodes amplify each other.
Shard Capacity
Enough slots to hold QMult-augmenting shards without blocking triggers.

Skill Priority

  1. QMult per-flip passives — Always-on. The engine.
  2. Node trigger coverage — Each % directly increases base score.
  3. Shard efficiency — Nodes increasing max shard slots.
  4. Post-loss recovery — Losing still counts for node triggers.

⚠️ Reality Check
  • Requires fully unlocked Medic tree. Many hours of progression.
  • Geometric placement matters. Mispositioned nodes break the chain.
  • Impossible-tier items that boost Medic passives are rare.
  • This build is endgame only. Don't try as a beginner.
