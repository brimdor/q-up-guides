✅ How Recycling Works
Buying items then recycling them converts Gold → Gems. Up to 6 items at once. Specific groupings grant set bonuses like Balatro poker hands. One Target Set per session is permanently doubled after the first fulfillment.

🎯 The Target Set Rule
Every recycling session highlights one Target Set giving double bonus. Fulfill it once → that set's value permanently increases. Early: go for easy targets (One Pair, PVP). Late: chase Anomaly and Typical.
⚠️ Percentages Are Mutable
Percentages shown are base. Target Set fulfillment permanently raises them. Use as benchmarks, not fixed rules.

Efficient Gold → Gem Cycle

  1. Buy 6 cheap Commons/Uncommons during a shop sale
  2. Recycle them as a set (even basic pairs give +50–200%)
  3. Re-invest Gems into QMult upgrades
  4. Run matches with higher QMult → earn more Gold → repeat
